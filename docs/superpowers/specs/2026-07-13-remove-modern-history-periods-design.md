# 移除近现代历史时期设计

## 目标

从事件目录和“中华历史长河”中移除“中华民国”“中华人民共和国”两个时期，并移除 1912 年及之后的全部内置历史事件。

## 数据调整

- 从 `HISTORY_PERIODS` 删除两个近现代时期，使事件目录的时期导航止于“清”。
- 从 `DYNASTIES` 删除“中华民国”“中华人民共和国”两个长河区块。
- 从内置事件源删除 1912 年及之后的事件，使 `KEY_EVENTS` 的时间范围与时期数据保持一致。

## 一致性与边界

- 保留清朝及 1912 年以前的全部数据。
- 不在页面或组件中增加临时过滤逻辑，避免不同入口展示不一致。
- 通用历史事件说明生成能力保持不变，不限制调用方自行传入现代事件。

## 验证

- 更新数据总数与时期列表断言。
- 增加内置时期、朝代和事件均截止于 1912 年以前的回归断言。
- 运行相关单元测试、完整测试及构建检查。
